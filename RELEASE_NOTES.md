# 슈퍼로봇대전 GC 한글패치 v1.0.1

일본판 `GRWJD9` 전용 UI 이미지 한국어화 릴리스입니다.

## 변경 내용

- 대형·소형 UI와 타이틀 메뉴의 영문 이미지 452개를 한국어로 교체
- 일본판 원문의 의미를 기준으로 잘못 번역된 공격·상태·메뉴 항목 교정
- `PRESS START BUTTON`, `START`, `LOAD`, `CONTINUE`, `OPTION`을 한국어로 교체
- `Now Loading...`을 `불러오는 중...`으로 교체
- 모든 이미지에 런타임 대사 글꼴과 같은 맑은 고딕 Bold 사용
- 공유 타일로 인한 메뉴 글자 혼합을 막도록 SCR별 독립 렌더링·재패킹 적용

대사·이벤트 흐름, 일본판 기준 화자 순서, 주인공 이름과 이전 릴리스의
한국어 문자열 수정은 그대로 유지됩니다.

## 검증

- 한국어 문자열 페이로드 62,506개: 실패 0, 일본어 잔존 0
- UI·타이틀 SCR 484개: 한국어 교체 452개, 비문자 이미지 보존 32개
- 로딩 SPR: 8개 C4 텍스처만 변경, 팔레트·헤더·CEL 보존, 영문 픽셀 잔존 0
- 일본판 비교 대상 277개 파일: 변경 없음
- xdelta 역적용 결과 SHA-256 일치

## 적용

릴리스의 `SRW_GC_Korean_v1.0.1.xdelta`, `APPLY_PATCH.bat`,
`apply_patch.ps1`를 같은 폴더에 두고, 별도로 준비한 `xdelta3.exe`와 함께
사용하세요. 자세한 절차는 저장소의 README를 확인하세요.
