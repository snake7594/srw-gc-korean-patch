# 슈퍼로봇대전 GC 한글패치 v1.0.6

v1.0.5까지 기계번역 상태로 남아 있던 문장 19,614건을 일본어 원문과 전후 문맥에 대조해 다시 번역한 릴리스입니다. 게임 구조, UI 이미지, 글꼴, 정신기 교정, 도감 런타임 수정은 v1.0.5와 동일합니다. 실제로 `add00dat.bin`, `font.pak`, `opening.bnr`은 v1.0.5와 SHA-256이 같으며, 대사·메뉴·전투·시스템 텍스트를 담은 파일만 달라집니다.

## 번역 품질 개선

- 기계번역으로 남아 있던 텍스트 **19,614건**을 일본어 원문·화자·전후 문맥 기준으로 재번역
  - 시나리오 대사 12,079건
  - 전투 대사 6,911건
  - 메뉴·도감 495건
  - 실행 파일 내 시스템 문구 129건
- 의미가 뒤바뀌어 있던 오역 교정. 예를 들어 `実の息子`가 `열매의 아들`로, 노인의 1인칭 `ワシ`가 `독수리`로, `好きにしろ`가 `좋아해!`로 옮겨져 있던 문장을 각각 `친아들`, `나`, `마음대로 해라`로 수정
- 한 대사 안에서 뒤섞이던 존댓말과 반말을 화자와 관계에 맞게 통일하고, 조사 오류(`지구은` → `지구는`)와 직역투 정리
- 인물·로봇도감 설명 482건을 재번역. 예를 들어 `직정형의 시노부란충돌을 반복하고`처럼 문장이 무너져 있던 설명을 `직정적인 시노부와는 충돌을 거듭했다`로 교정
- 의미 없이 음차되어 있던 표현을 뜻이 통하는 한국어로 교정
- 인명·기체명·용어는 기존 확정 번역에서 만든 용어집에 맞춰 통일

## 검증

- 교정본 전량을 실제 빌드 인코더로 검증했습니다. 표시 폭 초과, 구조 토큰(`KK`, `<AA>`, `<FF>`, `<TT>`) 변형, 글꼴 코드북에 없는 글자, 일본어 잔존은 모두 0건입니다.
- 대사의 화자 이름 구간이 본문으로 덮어써지지 않았는지 12,582건 전수 감사했습니다. 손상 0건입니다.
- 도감 설명은 런타임 버퍼 규격인 최대 24열로 다시 배치했고, 줄 폭 초과·줄바꿈 표식 오류·끝 표식 누락 0건을 확인했습니다.
- 표본 220건을 별도 검수한 결과 오역·의미 왜곡(serious) 0건, 사소한 표현 문제 14건이었습니다.
- 빌드 산출물 독립 감사에서 텍스트 페이로드 62,506건 검증 실패 0건, 일본어 잔존 0건, 승인 외 실행 코드 변경 0건을 확인했습니다.
- 일본판 비대상 파일 277개는 원본과 동일합니다.
- xdelta 패치를 원본에 역적용해 결과 ISO의 SHA-256이 일치함을 확인했고, 배포되는 `apply_patch.ps1`을 실제로 실행해 재현을 검증했습니다.

## 재현성

이번 릴리스는 `tools/build_iso_from_maps.py`로 생성했습니다. 새 빌드를 만들기 전에 같은 도구로 v1.0.5를 그 번역 맵에서 다시 빌드해, 공개된 v1.0.5 ISO의 SHA-256(`7E9DA87B…`)과 바이트 단위로 일치하는 것을 먼저 확인했습니다. 따라서 v1.0.6과 v1.0.5의 차이는 번역 텍스트뿐입니다.

## 이전 버전 주의

- **v1.0.1**은 전투 화면의 동적 타일 번호를 손상할 수 있으므로 사용하지 마세요.
- **v1.0.3**은 인물·로봇도감에서 글자 깨짐과 잘못된 메모리 읽기가 발생해 철회했습니다.
- **v1.0.4**는 정신기 설명 23건이 다른 정신기에 연결되므로 v1.0.6으로 교체하세요.

## 알려진 제한

- 이 릴리스는 자동 감사와 표본 검수로 검증했으며, 에뮬레이터·실기에서의 전체 플레이 검증은 하지 않았습니다.
- 원문 그대로 두는 것이 맞다고 판단한 문장 약 1,800건은 기존 번역을 유지했습니다.

# 슈퍼로봇대전 GC 한글패치 v1.0.5

v1.0.4에서 정신기 설명 23건이 다른 정신기에 연결된 문제와 일부 축약·중복 표기를 수정한 릴리스입니다. 인물·로봇도감 런타임 수정, PDF 번역 품질 교정, 전투 유닛 그래픽 보호 방식과 일본판 이벤트 순서는 그대로 유지합니다.

## 정신기 표기 및 설명 수정

- 정신기 전체 명칭 30건, 축약 30건, 설명 29건을 일본판의 고정 ID와 일대일로 재검증
- 번뜩임 설명에 저격 효과가 표시되는 등 `r0003~r0025` 설명 23건이 연쇄 이동한 문제 수정
- 이미지처럼 표시되는 1글자 축약의 `자기→자`, `력→노`, `난→란` 오류 수정
- 실행 파일에 중복 저장된 `근성 도 근성→근성　초근성`, `카쿠란→교란`, `힘→기력`, `애→사랑` 표기 수정
- 보이는 설명 29건 전부를 최대 2줄·한 줄 23자·41바이트 이내로 정리하고 숨김 `？？？` 슬롯은 일본판 그대로 보존
- 원인이었던 정신기 설명 CSV 순번 매칭 결과를 안정 ID 교정으로 덮어쓰고 이름·축약·설명의 전체 배열을 검사하는 회귀 방지 도구 추가
- 정신기 관련 공통 UI 이미지 14건은 승인본과 픽셀 단위 일치, 필요한 한글 글리프 누락 0건 확인

## 인물·로봇도감 런타임 수정

- 인물도감 267건과 로봇도감 238건, 총 505건의 설명문을 일본판 도감 렌더러 규격에 맞춰 재배치
- 각 줄을 최대 24열·48바이트로 제한하고 ASCII 문자·숫자·공백을 2바이트 표시 셀로 정규화
- 줄바꿈 표식의 홀수 바이트 오프셋, 홀수 바이트 구간, 잘못된 16비트 문자 단위, 헤더 줄 수 불일치 및 끝 표식 누락 0건
- 문제 빌드 회귀 검사에서 2천 건 이상의 구조 오류를 검출하고, 수정 빌드에서는 동일 검사 오류 0건 확인
- 인물도감의 `기도・조타로` 오기를 `키도・죠타로`로 교정

## 번역 품질 교정

- 안정 ID 기준 실제 텍스트 2,778건 교정
- 초반 압사라스 실험장, 올리비 설득 장면 등 대사 95건 직접 교정
- `하야토`, `코우지`, `테츠야`, `아스카`, `진`, `츠토무`처럼 조사·호칭이 붙어 누락됐던 인명 표기 정리
- `모가미 중공`, `소울 세이버`, `다이오쟈`, `바쿠신거`, `수전기대`, `바쿠류오`, `사악수` 등 반복 용어 통일
- 다른 레코드 문장이 섞이거나 내부 placeholder만 제거됐던 도감 장문 13건을 일본어 원문에서 전면 재번역
- 번역 맵의 내부 placeholder와 일본어 문자 잔존 0건

## 시나리오 제목 이미지

- 시나리오 제목 39건을 PDF와 일본어 원문으로 재검토
- 이미 정확했던 2건을 제외한 독립 BMP 37개를 글꼴 제작에 사용한 굵은 맑은 고딕으로 다시 렌더링
- 에피소드 번호↔블록 번호 허용 목록과 원본 블록 SHA-256을 고정해 다른 UI 블록 오염 방지
- PDF 내부 표기가 충돌하는 경우 일본어 원문과 통용 표기를 우선하고, 남녀 주인공 이름은 각각 `아카츠키 아키미`와 `아카츠키 아케미`로 유지

## 바이너리 검증

- 일본판 런타임 기준 한국어 문자열 페이로드 62,506개: 실패 0, 일본어 잔존 0
- add01 대사 정렬·제어 토큰·종료 코드 오류 0
- 도감 505건: 최대 24열·48바이트, 비정렬 줄바꿈·홀수 구간·잘못된 문자 단위·버퍼 초과 0
- 원본 UI 아틀라스 타일 변경 0, 동적 타일 후보 1,383개 보존
- 일본판 비교 대상 파일 277개 변경 없음
- 알려진 영문판 패치 표식 0건
- 애플리케이션 헤더 없이 만든 xdelta의 역적용 결과가 최종 ISO SHA-256과 일치

## 적용

수정되지 않은 일본판 원본 ISO에 `SRW_GC_Korean_v1.0.5.xdelta`를 적용하세요. 이전 한글패치가 적용된 ISO에는 덧씌우지 마세요. 릴리스의 `APPLY_PATCH.bat`, `apply_patch.ps1`와 별도로 준비한 `xdelta3.exe`를 같은 폴더에 두고 사용합니다.
