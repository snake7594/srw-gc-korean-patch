# 슈퍼로봇대전 GC 한글패치 v1.0.2

전투 화면의 유닛 그래픽 손상을 수정한 긴급 릴리스입니다.
**v1.0.1은 사용하지 말고 v1.0.2로 교체하세요.**

## 원인과 수정

v1.0.1의 UI 재패커는 정적 SCR에서 보이지 않는 아틀라스 타일을 미사용
공간으로 판단해 전체 타일 번호를 다시 배치했습니다. 그러나 게임은 전투와
동적 메뉴에서 일부 타일을 번호로 직접 선택하므로, 한국어 메뉴 타일이 유닛
그래픽 자리에 표시될 수 있었습니다.

v1.0.2는 다음과 같이 수정했습니다.

- 원본 UI 아틀라스 타일 11,872개를 같은 번호와 바이트로 전부 보존
- 정적 SCR 미참조 비공백 타일 1,383개 전부 보존
- 비문자 SCR 32개를 바이트 단위로 보존
- 한국어 대상 SCR 452개만 아틀라스 뒤의 새 타일을 참조
- `Now Loading...` 한국어화와 기존 대사·메뉴 번역은 유지

블록 `438`은 10비트 타일 번호의 물리적 한계 때문에 전투 효과 문구를
굵은 맑은 고딕 8×8 글리프로 표시합니다. 글자는 이전 릴리스보다 작지만,
동적 전투 타일을 손상하지 않는 안전한 배치를 우선했습니다.

## 검증

- 기존 아틀라스 타일 변경: 0
- 동적 참조 후보 타일 보존: 1,383/1,383
- 한국어 대상 SCR 시각 왕복 실패: 0
- 한국어 문자열 페이로드 62,506개: 실패 0, 일본어 잔존 0
- 일본판 비교 대상 파일 277개: 변경 없음
- xdelta 역적용 결과 SHA-256 일치

## 적용

수정되지 않은 일본판 원본 ISO에 `SRW_GC_Korean_v1.0.2.xdelta`를
적용해야 합니다. v1.0.1이 적용된 ISO에는 덧씌우지 마세요. 릴리스의
`APPLY_PATCH.bat`, `apply_patch.ps1`와 별도로 준비한 `xdelta3.exe`를
같은 폴더에 두고 사용하세요.
